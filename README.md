## Installation & Setup

### Clone the Repository
```
git clone git@github.com:git@github.com:Sergey2829/ImFeelingLucky.git

cd ImFeelingLucky
```

### Install dependencies

```
composer install

npm install
```

### Environment configuration

```
cp .env.example .env
```


### Database setup

```
php artisan migrate
```

### Run the project

```
php artisan serve

npm run dev
```

Visit: http://localhost:8000
